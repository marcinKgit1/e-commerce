## 🌟 O Projekcie

**TwójSklep** to nowoczesna, lekka i wydajna aplikacja e-commerce zbudowana w oparciu o najnowszy stos technologiczny. Projekt został poddany szczegółowemu audytowi pre-deployment, co zaowocowało wysoką jakością kodu, optymalizacją pod kątem SEO oraz pełną dostępnością (A11y).

## 🚀 Kluczowe Funkcjonalności

- 💳 **Zarządzanie Koszykiem**: Dodawanie, usuwanie i dynamiczna zmiana ilości produktów.
- 🌙 **Dark Mode**: Pełne wsparcie dla motywu ciemnego i jasnego z automatycznym zapamiętywaniem preferencji.
- 🌍 **Wielojęzyczność**: Wsparcie dla języka polskiego i angielskiego (i18n).
- 🔍 **SEO & Social Ready**: Kompletne metatagi oraz obsługa OpenGraph dla mediów społecznościowych.
- ♿ **Dostępność (A11y)**: Pełna obsługa czytników ekranowych dzięki atrybutom `aria-label`.
- ⚡ **Wysoka Wydajność**: Optymalizacja re-renderów przy użyciu `useMemo` i `useCallback`.
- 🛡️ **Stabilność**: Implementacja `Error Boundary` chroniąca przed awariami UI.

## 🛠️ Stos Technologiczny

- **Frontend**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vite.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animacje**: [Motion](https://motion.dev/)
- **Ikony**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)

## 💻 Uruchomienie Lokalne

1. **Sklonuj repozytorium:**
   ```bash
   git clone https://github.com/marcinKgit1/e-commerce.git
   cd e-commerce
   ```

2. **Zainstaluj zależności:**
   ```bash
   npm install
   ```

3. **Uruchom serwer deweloperski:**
   ```bash
   npm run dev
   ```

Aplikacja będzie dostępna pod adresem: `http://localhost:3000`
